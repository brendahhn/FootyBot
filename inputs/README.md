# inputs/

Drop user-provided raw research here as it's gathered — it feeds Phase 1 research, it isn't
itself the output.

- `findings/` — manual notes/screenshots/links the user has already gathered (e.g. the
  league settings screenshots used to write `CONTEXT.md`).
- `twitter-likes/` — the Twitter "likes" export, once available. Format unknown as of
  2026-06-30 (could be a JS/JSON archive file, a list of URLs, or screenshots) — drop
  whatever comes out and we'll parse it then.
